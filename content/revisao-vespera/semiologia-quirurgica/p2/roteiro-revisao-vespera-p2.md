# Roteiro - Revisao de Vespera P2 - Semiologia Quirurgica

> [!warning] Uso correto
> Este roteiro e para **executar a revisao em ordem**, sem abrir materiais aleatorios. Se travar, volte ao arquivo de vespera do tema. So abra o resumo completo quando o arquivo de vespera nao resolver.

---

---

## 2. Como Fazer Cada Bloco

1. Ler **Ideia Central em 3 Linhas**.
2. Ler **O Que Mais Cai**.
3. Ler **Pegadinhas e Erros Comuns**.
4. Responder as **10 Perguntas Rapidas** sem olhar as respostas.
5. Conferir as **Respostas Curtas**.
6. Marcar mentalmente o checklist.
7. Se errou mais de 3 perguntas, reler aquele tema antes de seguir.

> [!tip] Regra de avanco
> So passe para o proximo tema se acertar pelo menos **7/10 perguntas rapidas**.

---

## 3. Pontos Que Precisam Sair Sem Pensar

| Tema | Resposta que precisa estar automatica |
|---|---|
| P2-T01 | Hemoperitoneo = sangue esteril, pouco irritante, predomina shock. |
| P2-T01 | Geneau de Mussy cai pelo nome; Jobert positivo contraindica LPD. |
| P2-T02 | Epigastralgia aguda exige descartar IAM inferior com ECG/enzimas. |
| P2-T02 | Moynihan: gastrica piora com alimento; duodenal melhora e doi em jejum/noite. |
| P2-T03 | Apendicitis: dor primeiro, febre depois. Psoas = retrocecal; obturador = pelviano. |
| P2-T03 | Alvarado: dor em FID e leucocitose valem 2 pontos. |
| P2-T04 | FII comeca pelo sigmoide; diverticulitis complicada pede TC, nao colonoscopia imediata. |
| P2-T04 | Fecaluria + neumaturia = fistula colovesical. |
| P2-T05 | Murphy positivo = interrompe inspiracao, nao apenas dor. |
| P2-T05 | Charcot = febre + ictericia + dor; Reynolds = Charcot + shock + alteracao neurologica. |

## 4. Rodada de Perguntas por Ordem

### P2-T01 - Abdomen agudo hemorragico

- [ ] Por que o sangue irrita pouco o peritonio?
- [ ] Qual a causa nao traumatica mais comum?
- [ ] O que e Geneau de Mussy?
- [ ] Quais sao as contraindicacoes relativas do LPD?
- [ ] Por que Jobert positivo muda a conduta?

### P2-T02 - Dolor en epigastrico

- [ ] Qual diagnostico descartar primeiro?
- [ ] Como separar IAM de ulcera?
- [ ] O que e ritmo de Moynihan?
- [ ] O que indica vomito blanco?
- [ ] O que e Tumor de Virchow?

### P2-T03 - Fosa iliaca derecha

- [ ] Qual e a Cronologia de Murphy?
- [ ] O que significa febre primeiro?
- [ ] Quais manobras indicam retrocecal e pelviano?
- [ ] Quais itens valem 2 pontos no Alvarado?
- [ ] Como diferenciar adenitis mesenterica?

### P2-T04 - Fosa iliaca izquierda

- [ ] Qual a patologia principal da FII?
- [ ] Como nasce o diverticulo falso?
- [ ] Quando nao fazer colonoscopia?
- [ ] O que significa fecaluria/neumaturia?
- [ ] Como diferenciar SII de diverticulitis?

### P2-T05 - Sindrome biliar / sx. del HD

- [ ] Diferencie colico biliar e colecistitis.
- [ ] Qual e a tecnica correta de Murphy?
- [ ] Qual e a Triada de Charcot?
- [ ] O que acrescenta Reynolds?
- [ ] O que define TG18 Grado III?
